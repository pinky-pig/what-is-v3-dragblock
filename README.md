## Git commit

- Chore : 杂活 -- 构建管理器配置等，代码无变动。  
- Feat : 新功能  
- Fix : Bug 修复  
- Style : 修改代码格式、补充分号等
- Docs : 文档修改，注释更改
- Refactor : 代码重构
- Test : 测试添加、测试重构等，代码无变动
- Revert : 恢复先前的提交
